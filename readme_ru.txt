=== Pokupo Woocommerce gateway ===
Tags: pokupo, woocommerce
Requires at least: 4.6
Tested up to: 5.3.2
Requires PHP: 5.2.4 or later

== Описание ==

Pokupo это универсальное решение для создания интернет магазина и платежных ссылок, 
с более чем 30 методами оплаты и бесплатной онлайн кассой.

Данный модуль представляет собой платежный шлюз для WordPress + WooCommerce к мерчанту Pokupo.

== Установка ==

1. В админке Wordpress перейдите в `Плагины->Добавить новый`, нажмите кнопку `Загрузить плагин`
2. Выберите скачанный архив с модулем и нажмите `Установить`
2.1 В случае проблем с установкой скопируйте содержимое архива в папку вашего сайта `wp-content/plugins`
3. Перейдите в `Плагины->Установленные`, найдите `Pokupo WooCommerce Gateway` и нажмите `Активировать`
4. Перейдите в `WooCommerce->Settings->Checkout`
5. Выберите Pokupo
6. В поле `Shop ID` введите 'Идентификатор магазина для CMS' из Настроек магазина в личном кабинете Покупо
7. В поле `Notification Password` введите 'Пароль уведомления' из Настроек магазина в личном кабинете Покупо
8. В Настройках магазина в личном кабинете Покупо в поле URL уведомлений кажите адрес из строки `Notification URL`
